```bash

```

```bash

```

4. Configure environment variables:
```bash
cp .env.example .env
php artisan key:generate
```

5. Set up the database:
```bash
php artisan migrate
php artisan db:seed # if you want to seed the database with sample data
```

6. Start the development servers:

For Laravel:
```bash
php artisan serve
```

For React/Vite:
```bash
npm run dev
```
